# 小红书拼贴海报编辑器

一个本地静态原型，用输入文字生成参考图风格的拼贴海报，并支持拖拽微调、上传图片、修改样式和导出 PNG。

## 本地运行

```bash
python3 -m http.server 5173
```

然后打开：

```text
http://localhost:5173
```

## 第一版功能

- 根据输入文本生成无内置图片的拼贴布局。
- 生成时按文字高度流式排版，避免初始文字块互相遮挡。
- 文本、便签、图片元素可拖拽。
- 选中元素后可调整文字、字号、宽度、颜色、背景透明度。
- 支持上传本地图片作为拼贴素材。
- 支持整体字体切换和 5 套配色预设。
- 导出 1080 x 1440 PNG。
